# Birthday Website

A simple birthday website for Juvana, built with plain HTML, CSS, and JavaScript.

## Files

- `birthday.html` - page structure
- `birthday.css` - styling and responsive layout
- `birthday.js` - popup interaction and hero confetti

## Preview Locally

Open `birthday.html` in your browser.

## Live Site

After GitHub Pages is enabled for this repository, the site URL will be:

`https://phebronia.github.io/birthday/`
